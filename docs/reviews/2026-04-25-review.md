### Medium

- **`.claude/settings.json:6-28` — security / permissions.** Wildcard glob
  permissions (`Bash(git *)`, `Bash(npm *)`, `Bash(cargo *)`, `Bash(brew *)`,
  …) authorise *any* subcommand, including destructive or remote-publishing
  ones (`git push --force`, `npm publish`, `brew uninstall`, `cargo
  publish`). For an unattended review job this is an unnecessarily large
  blast radius.
  **Fix:** narrow to the actually-used subcommands, e.g.
  `Bash(git status)`, `Bash(git add)`, `Bash(git commit)`,
  `Bash(git push)`, `Bash(git diff *)`, `Bash(git log *)`,
  `Bash(pnpm install)`, `Bash(pnpm test *)`, `Bash(pnpm run *)`,
  `Bash(cargo test *)`, `Bash(cargo build *)`. Drop package managers
  not actually used in the repo (gem/bundle/composer/mvn/gradle/brew/pip/
  poetry — Driveline is Rust + pnpm).
